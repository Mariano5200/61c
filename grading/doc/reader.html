Getting Started

This tutorial assumes prior experience with SSH. If you have not previously used SSH, please refer to the Instructional Support Group’s ssh documentation.

Grading

Making a Directory

You’ll want to keep the grading directories separate. Start by making a directory for your current assignment, and cd-ing into it:

cs123-ra% mkdir hw1
cs123-ra% cd hw1

Making a Roster

Before you start grading you will want to create the file you will be entering grades into. Run:

cs123-ra% make-roster -o hw1roster hw1

This will create a file called hw1roster for the assignment hw1. Substitute the assignment and file names as necessary. The contents of the file will be the logins and names of all your assigned students for the given assignment.

hw1roster:

cs123-aa Douglas_Adams 
cs123-ab Isaac_Asimov

Retrieving Assignments

Once you’ve created your roster, you may begin grading assignments. There are two ways to accomplish this:

Using get-submissions

cs123-ra% get-submissions hw1
cs123-ra% ls
cs123-aa.200901140126@  cs123-ab.200901162224@  hw1roster

As you can see in the previous command (and output), get-submissions will create links in the current directory to student submissions. To grade each of these submissions, run the command lookat:

 cs123-ra% lookat cs123-aa

This will look in the current directory and unpack the latest submission for cs123-aa into the directory LOOK.

Using get-subm

cs123-ra% mkdir LOOK
cs123-ra% cd LOOK
cs123-ra% get-subm hw1 cs123-aa

This retrieves the submission for cs123-aa and unpacks it in the current directory.

Entering Grades

The grading file you created earlier, in our example hw1roster, should be filled in, with spaces separating four fields, in order: login, name, points, and comments. Spaces do not matter for the comment field. For example:

cs123-aa Douglas_Adams 42 Almost but not entirely unlike good code.
cs123-ab Isaac_Asimov 4920 Your code reads like a robot wrote it.

When you have finalized your grades, save the modified hw1roster file. You may now double-check your scores, or if you are certain of their correctness, run enter-grades:

cs123-ra% enter-grades -f hw1roster hw1

This will flag any errors in your roster file, and once you have corrected these, your grades will be updated and will go live. Alternatively, you may manually enter grades by omitting the -f flag:

cs123-ra% enter-grades hw1
Enter grades one at a time.  End with a login of '.'
Login:

Enter the login, and you will be prompted for the grade and comments.

This method has the added advantage of entering partners’ grades, however as you will probably already be keeping track of partners (denoted in the MY.PARTNERS file in project submissions) while filling out the hw1roster file, the first method is preferred for bookkeeping purposes.
